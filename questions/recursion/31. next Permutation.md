// 1. find pivot: first nums[i] < nums[i+1] from right
// 2. swap pivot with smallest element > pivot in suffix
// 3. reverse suffix to get next permutation

Example: 

1 5 8 7 6 4 3

nums[i] < nums[i+1]

4 ≥ 3
6 ≥ 4
7 ≥ 6
8 ≥ 7
5 < 8  ← pivot

pivot = 5, suffix i descending: 8 7 6 4 3

Because the suffix is descending, the correct element will be the first number from the right that is > pivot.

Check from right:

3  ≤ 5
4  ≤ 5
6  > 5  ← swap candidate

Swap:

1 6 8 7 5 4 3

Suffix before reverse:

8 7 5 4 3

Reverse:

3 4 5 7 8

Final result:

1 6 3 4 5 7 8

TC: O(N)
SC: O(1)
``` cpp
class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        
        int n = nums.size(), i = n - 2;
        while(i >= 0 && nums[i] >= nums[i+1]) i--;

        if(i >= 0){
            int j = n - 1;
            while(nums[j] <= nums[i]) j--;

            swap(nums[i], nums[j]);
        }

        reverse(nums.begin()+i+1, nums.end());
    }
};
